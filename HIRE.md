## Common engagement shapes

### Fixed-scope build
Best for: "I need a booking site / invoice tool / chat app / X."
You get: Full build (code + tests + Docker + CI), source transfer to your GitHub org, deployment to your domain, 30-day support window.
Pricing: Fixed quote after a 30-min discovery call.

### Part-time retainer
Best for: Ongoing dev work or maintenance on an existing Django app.
You get: 15–25 hrs/week, predictable hourly rate, async daily standup, monthly invoicing, weekly check-in calls.
Pricing: Hourly, monthly minimum.

### Code review
Best for: "Audit my Django app before I ship."
You get: PR-style line-by-line review with security + performance + maintainability flags, plus an executive-summary doc.
Pricing: Per kLOC reviewed.

### Architecture consult
Best for: Pre-build planning ("what should we build, how, with what tech?")
You get: 1–3 hours of design thinking on call, deliverable is a written plan with tradeoffs, model sketches, and risk callouts.
Pricing: Fixed per consult.
